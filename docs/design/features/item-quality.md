# 道具品质系统

## 概述

所有可购买、可合成的道具（武器、扩展）统一使用 **6 档品质**。品质通过 UI **底图颜色**区分，玩家无需读文字即可识别稀有度。

## 品质档位

| 档位 | 名称（示例） | 底图色（建议） | 数值系数（示例） | 商店权重（前期） |
|------|--------------|----------------|------------------|------------------|
| 1 | 普通 | `#9E9E9E` 灰 | ×1.00 | 50% |
| 2 | 优秀 | `#4CAF50` 绿 | ×1.15 | 28% |
| 3 | 精良 | `#2196F3` 蓝 | ×1.35 | 14% |
| 4 | 史诗 | `#9C27B0` 紫 | ×1.60 | 6% |
| 5 | 传说 | `#FF9800` 橙 | ×1.90 | 1.8% |
| 6 | 神话 | `#F44336` 红 / 金红渐变 | ×2.25 | 0.2% |

> 色值可在 [visual-style.md](visual-style.md) 中与辉光、描边统一调色；上表为设计锚点。

## 视觉规范

### 道具卡片

```
┌─────────────────┐
│ ▓▓ 品质底图色 ▓▓ │  ← 整卡或边框条使用品质色
│    [图标]        │  ← 极简多边形图标
│    道具名称      │
│    简短描述      │
└─────────────────┘
```

- **底图**：品质色填充或渐变，透明度约 80%~100%。
- **图标**：中性色或略偏品质色，避免与底图混淆。
- **辉光**：品质 ≥4 时增加外发光，强度随品质递增（shader 实现）。
- **文字**：名称白色，描述浅灰；神话可加轻微描边动画。

### 战斗内掉落（若实现）

地面掉落物使用**小菱形/六边形** + 品质色 + 弱辉光，与商店卡片同源色板。

## 数值应用

品质系数作用于：

- 武器基础伤害、射速、弹数等
- 扩展效果幅度（如「燃烧伤害 +X」）
- 合成后的成长倍率（高品质合成收益更高）

公式示例：

```
final_damage = base_damage × quality_multiplier[quality] × synth_bonus
```

## 合成与品质

- 仅**同品质**可合成（见武器系统）。
- 合成不改变品质档位，只提升该档位内的数值。
- 升品途径（若未来加入）：专用道具或神话合成仪式，非首期。

## 数据定义

```json
{
  "quality": 1,
  "quality_name_key": "QUALITY_COMMON",
  "color": "#9E9E9E",
  "multiplier": 1.0,
  "glow_strength": 0.0
}
```

`glow_strength`：0（1~3 档）→ 0.3（4）→ 0.6（5）→ 1.0（6），供 UI 与 shader 读取。
